# 회계·딜·택스·AI 브리핑

회계, 인수합병(딜), 세무, AI 네 분야의 핵심 뉴스를 매일 정리해서 보여주는 정적 브리핑 페이지입니다.

## 이 프로젝트는

- 국내 회계법인 동향, M&A/딜 소식, 세무·세제 이슈, AI 산업·정책 뉴스를 하루 12건(분야별 3건)으로 압축해서 보여줍니다.
- 각 기사 카드는 언론사명, 날짜, 2~3줄 요약과 함께 원문 기사로 바로 연결되는 링크를 제공합니다.
- 상단에는 그날 기사 중 가장 눈에 띄는 수치 3개를 뽑은 통계 하이라이트가 있습니다.

## 디자인

PwC의 기업 지속가능성 리포트(FY24)에서 볼 수 있는 시각 언어 — 세리프 대제목, 오렌지·앰버·레드 계열 그라데이션 카드, 컬러 섹션 구분바, 통계 콜아웃 박스, 좌하단 워드마크 — 를 참고해 만들었습니다.

다만 **PwC의 로고나 브랜드명은 사용하지 않았습니다.** 이 페이지는 특정 회계법인이나 기업의 공식 발행물이 아니라 개인이 뉴스를 큐레이션하기 위해 만든 것으로, 참고한 것은 어디까지나 시각적 스타일(색상, 레이아웃, 타이포그래피)뿐입니다.

라이트/다크 테마를 모두 지원하며, CSS 커스텀 프로퍼티로 색상 토큰을 관리합니다.

## 콘텐츠는 어떻게 만들어지나

기사는 매일 다음 순서로 수집·검증됩니다.

1. 네이버 뉴스(news.naver.com)는 브라우징 정책상 직접 접근할 수 없어서, 대신 원문 언론사 사이트(한국경제, 조세일보, 세계일보, 뉴스핌, 한국세정신문, 이데일리, 딜사이트, 한국M&A경제신문, AI타임스, 파이낸셜뉴스 등)를 웹 검색으로 찾습니다.
2. 후보 기사는 반드시 원문을 직접 열어 제목·언론사명·날짜·핵심 내용을 확인한 뒤에만 채택합니다. 검색 결과 요약만 보고 넣지 않습니다.
3. 확인된 내용만 회계 3건·딜 3건·택스 3건·AI 3건, 총 12건으로 정리합니다.

이 방식이라, 발행되지 않은 내용을 지어내지 않는 대신 완전한 실시간 뉴스 피드는 아니고 **하루 단위 스냅샷**입니다.

## 직접 검색 (Live Search)

4개 섹션 스냅샷과 별개로, 페이지 상단에 원하는 키워드로 네이버 뉴스를 실시간 검색하는 기능이 있습니다.

- `api/search.js` — 네이버 "뉴스 검색" 오픈API를 호출하는 Vercel 서버리스 함수. API 키는 이 함수 안에서만 쓰이고 브라우저로 노출되지 않습니다.
- 페이지의 검색창은 이 함수(`/api/search?q=...`)를 호출해 결과를 같은 카드 디자인으로 보여줍니다.

### 설정 방법

1. [developers.naver.com](https://developers.naver.com/apps/#/register)에서 애플리케이션을 등록하고 "검색" API 사용을 추가해 **Client ID / Client Secret**을 발급받습니다. (본인 계정으로 직접 진행해야 합니다.)
2. Vercel 프로젝트의 **Settings → Environment Variables**에서 다음 두 값을 추가합니다.
   - `NAVER_CLIENT_ID`
   - `NAVER_CLIENT_SECRET`
3. 환경변수를 추가한 뒤 재배포(Redeploy)하면 검색창이 동작합니다.

이 키를 설정하지 않으면 검색창은 "API 키가 설정되지 않았습니다"라는 안내 메시지를 보여줍니다. 검색 기능은 서버리스 함수가 필요해서 GitHub/Vercel 배포본에만 있고, claude.ai 아티팩트 스냅샷에는 포함되어 있지 않습니다.

## 자동 업데이트

매일 아침 8시(한국 시간)에 예약된 작업이 자동으로 실행되어:

- 그날의 회계·딜·택스·AI 기사 12건을 새로 수집하고
- `index.html`을 갱신해 이 저장소에 커밋·푸시하고
- 동일한 내용을 [claude.ai 아티팩트](https://claude.ai/code/artifact/1d796c91-8a84-4a2c-a76d-ff3f0739d2bc)에도 반영합니다.

이 저장소가 Vercel에 연결되어 있다면, 매일의 푸시가 자동으로 새 배포를 트리거합니다.

## 배포

`index.html` 하나로 구성된 순수 정적 페이지라 별도 빌드 과정이 필요 없습니다.

Vercel에서 이 저장소를 Import할 때:
- **Framework Preset**: `Other`
- **Build Command**: 비워두기
- **Output Directory**: 비워두기
- **Install Command**: 비워두기

그대로 Deploy를 누르면 배포됩니다.

## 면책

이 페이지는 각 언론사 원문을 요약해 개인이 구성한 뉴스 큐레이션이며, 특정 회계법인·기업과 제휴하거나 그들의 공식 발행물이 아닙니다. 정확한 내용은 각 기사의 원문 링크를 통해 확인해 주세요.
